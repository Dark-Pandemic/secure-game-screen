# secure-game-screen
Game screen with basic security features as well as logging 
A secure and interactive login system for a game built with Python, Pygame, and SQLite, featuring user registration, authentication, brute-force protection, and a polished UI. This project demonstrates practical cybersecurity concepts combined with game development.
Features

User Registration & Login – securely register and login users.

Password Hashing – passwords are stored safely using hashes.

Brute-Force Protection – failed login attempts lock the account.

Security Logging – logs suspicious activity for auditing.

Polished Pygame UI – active input box highlight, labels, and buttons.

Game Launch – successful login leads to a game screen.
